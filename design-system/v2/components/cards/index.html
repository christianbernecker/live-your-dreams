React/Next.js Integration

// components/Card.tsx
import { ReactNode, HTMLAttributes } from 'react';

interface CardProps extends HTMLAttributes {
  variant?: 'default' | 'featured' | 'glassmorphism';
  size?: 'small' | 'medium' | 'large';
  interactive?: boolean;
  loading?: boolean;
  children: ReactNode;
}

export const Card = ({
  variant = 'default',
  size = 'medium',
  interactive = false,
  loading = false,
  className = '',
  children,
  ...props
}: CardProps) => {
  const cardClasses = `
    lyd-card 
    ${variant !== 'default' ? variant : ''} 
    ${size !== 'medium' ? size : ''}
    ${interactive ? 'interactive' : ''}
    ${loading ? 'loading' : ''}
    ${className}
  `.trim();

  return (
    <div className={cardClasses} {...props}>
      {children}
    </div>
  );
};

// Card sub-components
export const CardHeader = ({ children, className = '', ...props }) => (
  <div className={`lyd-card-header ${className}`} {...props}>
    {children}
  </div>
);

export const CardBody = ({ children, className = '', ...props }) => (
  <div className={`lyd-card-body ${className}`} {...props}>
    {children}
  </div>
);

export const CardFooter = ({ children, className = '', ...props }) => (
  <div className={`lyd-card-footer ${className}`} {...props}>
    {children}
  </div>
);

export const CardTitle = ({ children, className = '', ...props }) => (
  <h3 className={`lyd-card-title ${className}`} {...props}>
    {children}
  </h3>
);

API Reference

Class Description
.lyd-card Base card class with premium styling and animations

.lyd-card-title Card title with consistent typography
.lyd-card-subtitle Secondary title or description
.small Small card size (max-width: 300px)
.large Large card size (max-width: 600px)

.lyd-card-badge Notification badge positioned on card

Accessibility & Best Practices

Keyboard Navigation

  • Tab to navigate between interactive cards
  • Enter/Space to activate card actions
  • Arrow keys for card grids
  • Focus indicators on all interactive elements

Screen Reader Support

  • Semantic HTML structure
  • ARIA labels for complex cards
  • Meaningful heading hierarchy
  • Alt text for card images

Visual Accessibility

  • WCAG AA contrast ratios
  • Clear focus indicators
  • Sufficient color differentiation
  • Readable typography scales

Interactive States

  • Hover feedback for all interactions
  • Loading states for async actions
  • Error state handling
  • Success confirmation feedback

Typography Scale Integration

Diese Card-Implementation nutzt die Typography Scale für 100%ige Konsistenz:

  • Card Title: var(--font-size-lg) (18px) - Prominente Überschriften
  • Card Subtitle: var(--font-size-sm) (14px) - Sekundäre Informationen
  • Card Text: var(--font-size-base) (16px) - CI-Standard
  • Property Price: var(--font-size-2xl) (24px) - Wichtige Zahlen
  • Badge Text: var(--font-size-xs) (12px) - Kompakte Labels

Real Estate Optimization

Speziell optimiert für Immobilien-Anwendungen:

  • Property Cards: Preis, Zimmer, Fläche, Baujahr strukturiert dargestellt
  • Agent Cards: Professionelle Kontaktinformationen und Credentials
  • Statistics Cards: KPIs und Marktdaten übersichtlich präsentiert
  • Image Handling: Optimierte Hover-Effekte für Property-Bilder
  • Status Badges: Verfügbarkeit und Immobilien-Status
