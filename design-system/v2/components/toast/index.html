Accessibility & Best Practices

Keyboard Navigation

  • Tab to navigate between buttons
  • Space or Enter to activate
  • Esc to cancel (in dialogs)

Screen Reader Support

  • ARIA live regions announce toast content
  • Descriptive labels for all controls
  • State announcements for changes

Timing Controls

  • Users can pause auto-dismiss by hovering
  • Focus prevents auto-dismiss
  • Critical errors persist until dismissed

High Contrast

  • Toast colors meet WCAG AA contrast ratios
  • Clear focus indicators
  • Sufficient touch targets (44x44px)

Mobile Optimized

  • Touch-friendly close buttons
  • Responsive positioning for mobile devices
  • Minimum 44px touch targets

Error Handling

  • Critical errors persist until manually dismissed
  • Clear error messages
  • Visual and audio feedback
