LYD Design System V2 - Component Overview

Welcome to the Live Your Dreams Design System. This comprehensive library provides all the essential components for building modern, luxury real estate applications. Each component is crafted with attention to detail, incorporating our signature gradients, premium typography, and glassmorphism effects.

Our design system emphasizes consistency, accessibility, and a premium user experience that reflects the luxury segment we serve.
