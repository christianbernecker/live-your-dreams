Interactive States

Hoverable Card

Hover over this card to see the elevation effect.

Clickable Card

This card is clickable and shows pointer cursor.

Hoverable + Clickable

Combined hover and click effects for maximum interactivity.

Properties

Property Type Default Description
variant string default Card style variant: default, elevated, glass, outlined
hoverable boolean false Enables hover elevation effect
clickable boolean false Shows pointer cursor and click states
padding string medium Internal padding: small, medium, large, none

Next.js Usage

// app/properties/page.tsx
'use client';

import { useEffect } from 'react';

export default function PropertiesPage() {
  useEffect(() => {
    import('@/lib/lyd-design-system');
  }, []);

  const handleCardClick = (propertyId) => {
    router.push(\`/properties/\${propertyId}\`);
  };

  return (
    <div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-6">
      {properties.map(property => (
        <lyd-card 
          key={property.id}
          variant="glass" 
          hoverable 
          clickable
          onClick={() => handleCardClick(property.id)}
        >
          <img 
            src={property.image} 
            alt={property.title}
            style={{
              width: '100%',
              height: '200px',
              objectFit: 'cover',
              borderRadius: '12px',
              marginBottom: '16px'
            }}
          />
          
          <div slot="content">
            <h3>{property.title}</h3>
            <p>{property.location}</p>
            <div className="price">€{property.price.toLocaleString()}</div>
          </div>
        </lyd-card>
      ))}
    </div>
  );
}
