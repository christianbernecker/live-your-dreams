Developing

Technical documentation and integration guides for implementing the LYD Design System in your applications.

Getting Started

The LYD Design System is built with Web Components for maximum compatibility and performance. It integrates seamlessly with Next.js applications through dynamic imports. 

// Install dependencies
npm install @liveyourdreams/design-system

// Import in your Next.js app
import { useEffect } from 'react';

export default function MyPage() {
  useEffect(() => {
    import('@liveyourdreams/design-system');
  }, []);

  return (
    <div>
      <lyd-button variant="primary">
        Click me
      </lyd-button>
    </div>
  );
}

Web Components

All components are built as Web Components, providing framework-agnostic usage while maintaining excellent performance and encapsulation. 

// Direct HTML usage
<lyd-button variant="primary" size="large">
  Schedule Property Viewing
</lyd-button>

<lyd-input 
  variant="search" 
  placeholder="Search properties..."
  label="Property Search">
</lyd-input>

<lyd-card variant="glass" hoverable clickable>
  <div slot="content">
    Property content here
  </div>
</lyd-card>
