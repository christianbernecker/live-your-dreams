Grid

The LYD grid system is based on native CSS Grid for best performance, changing its available columns on mobile (6 content columns + 2 safe zone columns) and desktop view ≥ 760px viewport width (8-narrow/12-basic/14-extended/16-wide content columns + 2 safe zone columns). It has a size range between 320px and 2560px, using fluid sized columns and gaps.

Grid Variants

Variant Mobile Columns Desktop Columns Use Case
Narrow 4 + 2 safe 4 + 2 safe Forms, focused content
Basic 6 + 2 safe 12 + 2 safe Standard content, property cards
Extended 6 + 2 safe 14 + 2 safe Complex layouts, dashboards
Wide 6 + 2 safe 16 + 2 safe Wide content areas
Full 8 columns 18 columns Hero sections, full-width media

Usage

HTML Usage

<lyd-grid variant="basic" gap="medium">
  <lyd-grid-item span="6-4">
    <!-- Property Card 1 -->
  </lyd-grid-item>
  <lyd-grid-item span="6-4">
    <!-- Property Card 2 -->
  </lyd-grid-item>
  <lyd-grid-item span="6-4">
    <!-- Property Card 3 -->
  </lyd-grid-item>
</lyd-grid>

CSS Classes

/* Grid Container */
.lyd-grid-basic {
  display: grid;
  grid-template-columns: 1fr repeat(6, minmax(0, 1fr)) 1fr;
  gap: clamp(16px, 3vw, 24px);
}

@media (min-width: 760px) {
  .lyd-grid-basic {
    grid-template-columns: 1fr repeat(12, minmax(0, 1fr)) 1fr;
  }
}

/* Grid Item */
.lyd-grid-item {
  grid-column: span 6; /* Mobile */
}

@media (min-width: 760px) {
  .lyd-grid-item {
    grid-column: span 4; /* Desktop */
  }
}

Do's and Don'ts

Do

  • ✓ Use the grid for consistent appearance
  • ✓ Align headlines on vertical grid lines
  • ✓ Set image focus within basic grid
  • ✓ Use appropriate grid variant for content

Don't

  • ✗ Scale elements bigger than 1920px max
  • ✗ Nest grids within each other
  • ✗ Break out of grid alignment
  • ✗ Use inconsistent gap sizes
