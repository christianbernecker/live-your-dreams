## 🎯 Was ist das LYD Design System?

Das **Live Your Dreams Design System** ist eine umfassende Sammlung von wiederverwendbaren Komponenten, Design-Tokens und Richtlinien, die entwickelt wurden, um konsistente und hochwertige Benutzererfahrungen in allen LYD-Produkten zu gewährleisten.

## 📦 Pakete

Das Design System besteht aus mehreren Paketen:

### `@liveyourdreams/design-system`
- **CSS-Module** und Design Tokens
- **Moderne CSS Features** (Custom Properties, Grid, Flexbox)
- **Responsive Breakpoints** und Utility-Klassen
- **Dark Mode Support**

### `@liveyourdreams/design-system-react`
- **React-Komponenten** mit TypeScript
- **Hook-basierte Architektur**
- **Kompositional APIs** für maximale Flexibilität
- **Form-Integration** mit React Hook Form

## 🛠️ Installation

```bash
# Installiere das gesamte Design System
npm install @liveyourdreams/design-system @liveyourdreams/design-system-react

# Oder nur die CSS-Basis
npm install @liveyourdreams/design-system
```

### CSS importieren

```css
/* In deiner Haupt-CSS-Datei */
@import '@liveyourdreams/design-system/dist/index.css';
```

### React-Komponenten verwenden

```tsx
import { LdsButton, LdsCard, LdsInput } from '@liveyourdreams/design-system-react';

function MyComponent() {
  return (
    <LdsCard>
      <h2>Meine Karte</h2>
      <LdsInput label="Name" placeholder="Gib deinen Namen ein" />
      <LdsButton variant="primary">Absenden</LdsButton>
    </LdsCard>
  );
}
```
